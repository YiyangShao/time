# Research Notes

## 目的

为“把时间拿回来”故事展馆记录可复用的调研结论、原始 reference 和口径说明。

当前原则：

- 先记录，再筛选是否进入 UI。
- 口径不一致的数据不能硬拼在一起。
- UI 层优先使用中国数据；全球数据只作为背景对照。

## 可直接进入当前版本 UI 的候选

### 1. 中国成年国民日均接触手机时长

- 结论：`2024 年中国成年国民日均接触手机时长约 108.76 分钟`
- 来源机构：中国新闻出版研究院
- 来源项目：第二十二次全国国民阅读调查
- 发布时间：2025-04-24
- 口径说明：这里是“日均接触手机时长”，更接近阅读调查里的接触终端时间，不等于完整意义上的屏幕点亮时长或移动互联网总使用时长。
- 是否适合直接展示：适合，但必须在旁边注明口径。
- 原始参考：
  - https://news.qq.com/rain/a/20250424A0587O00
  - http://www.news.cn/book/20250427/99fc0745c807495fbc430a803d1b4d31/c.html

### 2. 中国移动互联网月人均使用时长

- 结论：`2025 年月人均使用时长约 178.2 小时`
- 折算：约 `5.9 小时 / 天`
- 来源机构：QuestMobile
- 来源项目：2025 中国移动互联网秋季大报告
- 口径说明：这里是“移动互联网使用时长”，覆盖移动互联网整体使用，不应直接与阅读调查口径相加或比较为同一指标。
- 是否适合直接展示：适合，但必须说明它和“接触手机时长”不是一回事。
- 原始参考：
  - https://www.questmobile.com.cn/research/report/1985984354598359042
  - https://www.adquan.com/article/356612

## 当前作为背景参考，不直接在 UI 中强展示的候选

### 3. 全球移动使用背景

- 结论：DataReportal 的 `Digital 2025 Global Overview Report` 可以作为全球范围的数字使用背景参考。
- 来源机构：DataReportal / We Are Social / Meltwater
- 口径说明：当前还没有把其中某个明确数字清洗成与中国数据可直接并列展示的结论，因此先只保留为背景 reference。
- 是否适合直接展示：暂不直接展示数字，适合作为后续对照资料。
- 原始参考：
  - https://datareportal.com/reports/digital-2025-global-overview-report

## 当前版本 UI 建议写法

建议在最后一幕只展示两种信息：

1. 一个和主角故事相关的情绪性结果
2. 一张或两张小型研究卡片

建议文案方向：

- `研究显示，2024 年中国成年国民日均接触手机约 108.76 分钟。`
- `QuestMobile 的 2025 报告显示，中国移动互联网月人均使用时长约 178.2 小时。`

并在卡片底部补一句：

- `两条数据统计口径不同，仅用于帮助理解“时间被带走”的现实背景。`
